🎯 옥좌공성 흐름 (참고용)

바람의나라 클래식의 옥좌공성 진행 단계 시각화. 본인 점수가 어느 단계에서 누적됐는지 흐름을 이해하면 전략에 도움.

※ 각 단계 점수 분포는 일반 참고용. 정확한 룰은 운영진 공지 / 넥슨 공식 가이드 확인.

⚔️ 직업별 공성 역할

🗡️전사
탱커 / 격수

높은 체력으로 성문/봉인석 등 핵심 거점 사수. 격수 빌드는 단기흑수/혈투에서 활약.

🥷도적
올라운더 격수

은신/기동성으로 후위 침투 + 신수 단독 처치 가능. 솔플 점수 확보에 유리.

🔥주술사
딜 / 솔플 제왕

마력 기반 광역 딜. 봉인석/신수수호령 처리에 결정적. 가중치 점수 1위 다수.

🌿도사
지원 / 파티 필수

회복/버프로 파티 생존 보장. 본인 점수는 낮을 수 있으나 파티 전체 점수 견인.

🤝 연합 에티켓

  • 점수 인증 필수: 매주 신청 가능 요일에 자기 캐릭의 점수 등록
  • 정예 표시 정확히: ⭕ 참전 / ❌ 불참 / ⏳ 최대한 — 가중치에 직접 반영됨
  • 다계정 솔직히: 다계정 보유자는 모두 등록, 점수용병/참전용병은 비고에 명시
  • 문파장/부방장 존중: 문파 운영진의 결정에 우선 따름
  • 다른 문파 비방 금지: 연합 내 갈등은 문파장끼리 협의

❓ 자주 묻는 질문

점수가 잘못 등록됐어요

같은 닉네임으로 같은 날 다시 신청하면 자동으로 "갱신" 처리됩니다. siege.html 에서 ⟳ 갱신하기 버튼 사용.

"등록된 문원이 아닙니다" 경고가 떠요

문파장/부문파장이 명단에 등록해야 합니다. 본인 닉네임을 알려주세요.

OCR 이 점수를 못 찾았어요

"점수 영역 직접 선택" 버튼으로 숫자 부분만 드래그하면 정확도 올라갑니다. 그래도 안 되면 직접 입력하세요.

다른 문파 사람도 등록할 수 있나요?

아니요. 본인 소속 문파의 siege 페이지에서만 가능합니다. 홈에서 자기 문파를 선택해 주세요.

모바일에서도 다 되나요?

네. 카메라 촬영 → 즉시 OCR 가능. 사이트를 "홈 화면에 추가" 하면 앱처럼 사용 가능 (PWA).
