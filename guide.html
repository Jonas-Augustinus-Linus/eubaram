👑 옥좌공성 메커니즘 (옛날방식)

바람의나라 클래식 옥좌공성 은 특명/봉인석 같은 단계가 없습니다. 옥좌 점유자 = 성주 라는 단순하지만 잔혹한 룰.

0:00공성 시작
30분 동안종료 불가 구간 — 옥좌 점유/방어 누적
⏱️ 30:00최소 종료 시각
랜덤 종료 구간30분 이후 언제든 자동 종료
🏆종료 시점 옥좌 점유 문주 → 성주
  • 옥좌 점유 = 문주만 가능. 부방장/일반 문원은 옥좌에 앉아도 성주 결정에 영향 없음
  • 종료 시점은 랜덤. 정확히 30분에 끝나지 않음 — 30분 이후 시스템이 무작위 시점에 자동 종료
  • 종료 신호가 떠야 비로소 그 순간 옥좌 점유자가 성주로 확정
  • 종료 시점 옥좌가 비어있으면 미점령 (전 주 성주 유지 또는 공석)
  • 문주가 사망하면 부활 시간 동안 옥좌 점유 불가 — 치명적 약점

🎯 전략 카드

🏰옥좌 사수
기본 전략 · 문주 직접 점유

문주가 30분 이전부터 옥좌에 앉아 자리 잡기. 종료가 언제 뜰지 모르므로 30분 이후 매 순간 비울 수 없음. 문주 사망 시 즉시 옥좌 비어짐 → 다른 문파에게 점유 기회 헌납.

🛡️문주 보호
생존이 곧 승리

부방장/딜러진이 문주 주변에 진을 쳐 적 진입 차단. 도사는 회복/버프 풀가동. 문주가 살아남는 한 옥좌 점유 지속 → 종료 시점 일치 시 성주 확보.

🚪쪽문공성
정면 돌파 회피

적 본진이 정문에 몰려 있을 때 쪽문/우회로로 침투해 옥좌 기습. 정공법이 막힐 때 핵심 카드. 도적의 은신/기동성이 결정적.

⚔️문주 견제
적 옥좌 점유자 끌어내기

점유 중인 적 문주를 집중 공격해 처치 → 부활 시간 동안 옥좌 공백 발생. 그 틈에 본인 문주가 진입. 주술사 광역 + 도적 즉살 콤보.

⏱️시간 계산
30분 직후 집중 운영

0~30분: 진영 구축 + 옥좌 선점. 30분 직후: 매 순간 종료 가능하므로 옥좌에서 한 칸도 떨어지지 않기. 종료 신호 뜰 때 누가 앉아있냐가 전부.

🎭기만 전술
정보전 / 페이크

일부러 옥좌 비우고 적을 끌어들인 뒤 한꺼번에 처치. 또는 거짓 후퇴로 적 진영 분산 유도. 사전 디스코드 합의 + 신호 정해두기.

⚔️ 직업별 공성 역할

🗡️전사
옥좌 수비대 / 문주 방패

높은 체력으로 옥좌 주변 진영 사수 + 문주 호위 일순위. 적 문주가 옥좌 진입할 때 바디블록.

🥷도적
쪽문 침투 / 적 문주 즉살

은신/기동성으로 쪽문공성 핵심. 옥좌 점유 중인 적 문주 기습 처치 → 부활 시간 동안 옥좌 공백 강제.

🔥주술사
광역 견제 / 적 문주 노출

광역 마법으로 옥좌 주변 적 진영 분산. 적 문주를 호위에서 떼어내 도적/전사가 마무리할 수 있게 셋업.

🌿도사

문주에게 회복/버프 풀가동 — 문주가 죽지 않는 것 자체가 옥좌 점유 지속이고 곧 승리. 우리 문파의 옥좌공성 핵심.

🤝 연합 에티켓

  • 점수 인증 필수: 매주 신청 가능 요일에 자기 캐릭의 점수 등록
  • 정예 표시 정확히: ⭕ 참전 / ❌ 불참 / ⏳ 최대한 — 가중치에 직접 반영됨
  • 다계정 솔직히: 다계정 보유자는 모두 등록, 점수용병/참전용병은 비고에 명시
  • 문파장/부방장 존중: 문파 운영진의 결정에 우선 따름
  • 다른 문파 비방 금지: 연합 내 갈등은 문파장끼리 협의

❓ 자주 묻는 질문

옥좌공성에서 점수가 왜 중요한가요? 결국 옥좌 점유자가 성주잖아요?

맞습니다. 성주는 옥좌 점유 문주가 결정 하고, 등록 점수와 무관합니다. 점수 인증은 연합 내 활동 기록 용 — 누가 꾸준히 참전했고 누가 무임승차했는지 추적해 보상/제재 근거로 사용. 명예의 전당도 점수 기준 활동도 평가입니다.

30분 후 종료는 정확히 언제 떠요?

예측 불가입니다. 30분 직후 ~ 몇 분/십몇 분 사이 무작위. 30분 이후로는 매 초 옥좌가 비면 즉시 성주 빼앗길 수 있습니다.

문주가 죽으면 어떻게 되나요?

부활 시간 동안 옥좌 점유 불가. 즉시 다른 문파가 옥좌 점유 가능해집니다. 문주 보호는 옥좌공성의 가장 결정적 변수.

쪽문공성이 뭔가요?

정문이 적 본진에 막힌 상황에서 측면/뒷문(쪽문)으로 우회 침투. 적의 시선을 분산시키고 옥좌 기습이 핵심. 도적의 은신과 결합하면 매우 강력.

점수가 잘못 등록됐어요

같은 닉네임으로 같은 날 다시 신청하면 자동으로 "갱신" 처리됩니다. siege.html 에서 ⟳ 갱신하기 버튼 사용.

"등록된 문원이 아닙니다" 경고가 떠요

문파장/부문파장이 명단에 등록해야 합니다. 본인 닉네임을 알려주세요.

OCR 이 점수를 못 찾았어요

"점수 영역 직접 선택" 버튼으로 숫자 부분만 드래그하면 정확도 올라갑니다. 그래도 안 되면 직접 입력하세요.

다른 문파 사람도 등록할 수 있나요?

아니요. 본인 소속 문파의 siege 페이지에서만 가능합니다. 홈에서 자기 문파를 선택해 주세요.

모바일에서도 다 되나요?

네. 카메라 촬영 → 즉시 OCR 가능. 사이트를 "홈 화면에 추가" 하면 앱처럼 사용 가능 (PWA).
